# Alura 7 Days Of Code

## Python Web & API

Este repositório reúne projetos desenvolvidos em **Python** com foco em **desenvolvimento web** e consumo de **APIs**, como parte de uma iniciativa de aprendizado da plataforma Alura.

O objetivo é aplicar conceitos na prática, explorando integrações com serviços web e manipulação de dados.
